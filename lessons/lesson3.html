HTML Document Structure

An HTML document is made up of elements, each defined by tags. The basic structure includes <!DOCTYPE html>, <html>, <head>, and <body> tags.

Example

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Page</title>
  </head>
  <body>
    <header>Welcome!</header>
    <main>This is my first web page.</main>
  </body>
</html>

Tips

  • Always start with <!DOCTYPE html> for HTML5.
  • Use semantic elements like <header>, <main>, <footer> for better structure.
  • Close all tags properly.
