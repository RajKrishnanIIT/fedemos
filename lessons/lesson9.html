Lesson 7.2: Variables and Data Types

  • Declaring variables (let, const, var)
  • Primitive data types: string, number, boolean, null, undefined
  • Type coercion and checking types
  • Template literals and string methods

Example

let name = "Alice";
const age = 20;
var isStudent = true;
console.log(typeof name); // "string"
console.log(`Hello, ${name}!`); // Template literal

Lesson 7.3: Operators and Expressions

  • Arithmetic operators (+, -, *, /, %, **)
  • Comparison operators (===, !==, <, >, <=, >=)
  • Logical operators (&&, ||, !)
  • Assignment operators

Example

let x = 5, y = 3;
console.log(x + y); // 8
console.log(x > y); // true
console.log(x === 5 && y === 3); // true
x += 2; // x is now 7

Lesson 7.4: Functions

  • Function declarations vs. expressions
  • Parameters and arguments
  • Return statements
  • Arrow functions (ES6)
  • Scope and variable accessibility

Example

// Function declaration
function greet(name) {
  return `Hello, ${name}!`;
}
// Function expression
const add = function(a, b) {
  return a + b;
};
// Arrow function
const multiply = (a, b) => a * b;
console.log(greet('Bob'));
console.log(add(2, 3));
console.log(multiply(4, 5));
