Lesson 6.1: Responsive Design

  • Mobile-first approach
  • Media queries and breakpoints
  • Responsive units (em, rem, %, vw, vh)
  • Responsive typography and images

Example

body {
  font-size: 1rem;
}
@media (max-width: 600px) {
  body {
    font-size: 0.9rem;
  }
}
img {
  max-width: 100%;
  height: auto;
}

Lesson 6.2: CSS Animations and Transitions

  • Transition property for smooth changes
  • Transform property for 2D effects
  • Keyframe animations
  • Performance considerations

Example

button {
  transition: background 0.3s;
}
button:hover {
  background: #1976d2;
  transform: scale(1.05);
}
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
.box {
  animation: fadeIn 1s ease-in;
}
