Lesson 9.1: Introduction to the DOM

  • What is the DOM (Document Object Model)?
  • Selecting elements: getElementById, querySelector, querySelectorAll
  • Element properties: innerHTML, textContent, classList

Example

const title = document.getElementById('main-title');
const firstButton = document.querySelector('button');
const allItems = document.querySelectorAll('.item');
console.log(title.textContent);

Lesson 9.2: Modifying Elements

  • Changing content and attributes
  • Adding and removing CSS classes
  • Creating and removing elements
  • Modifying styles directly

Example

title.innerHTML = 'New Title';
title.classList.add('highlight');
title.classList.remove('highlight');
const newDiv = document.createElement('div');
document.body.appendChild(newDiv);
document.body.removeChild(newDiv);
title.style.color = 'red';

Lesson 9.3: Event Handling

  • Common events: click, submit, load, input
  • addEventListener method
  • Event objects and event properties
  • Preventing default behavior

Example

const btn = document.querySelector('button');
btn.addEventListener('click', function(event) {
  alert('Button clicked!');
  event.preventDefault();
  console.log(event.type);
});
