Lesson 2.1: Hyperlinks

  • Anchor tags and href attribute
  • Absolute vs. relative URLs
  • Link targets and accessibility
  • Email and telephone links

Example

<a href="https://developer.mozilla.org/" target="_blank">MDN Web Docs</a>
<a href="/lessons/lesson3.html">Go to Lesson 3</a>
<a href="mailto:someone@example.com">Email Us</a>
<a href="tel:+1234567890">Call Us</a>

Lesson 2.2: Lists and Navigation

  • Unordered lists (ul) and ordered lists (ol)
  • Description lists (dl, dt, dd)
  • Nested lists
  • Creating navigation menus

Example

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
  <li>First</li>
  <li>Second</li>
</ol>
<dl>
  <dt>HTML</dt>
  <dd>Markup language</dd>
</dl>

  • Image elements and alt text
  • Image formats and optimization
  • Audio and video elements
  • Responsive images basics

Example

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<video controls width="320" height="240">
  <source src="movie.mp4" type="video/mp4">
</video>
