Lesson 4.2: Selectors and Specificity

  • Element, class, and ID selectors
  • Descendant and child selectors
  • Pseudo-classes (:hover, :focus, :nth-child)
  • Specificity hierarchy and best practices

Example

/* Element selector */
h1 { color: green; }
/* Class selector */
.button { background: #1976d2; }
/* ID selector */
#main { padding: 2rem; }
/* Descendant selector */
nav ul li { list-style: none; }
/* Pseudo-class */
button:hover { background: #1565c0; }

  • Box-sizing property

Example

  width: 200px;
  padding: 20px;
  border: 2px solid #1976d2;
  margin: 10px;
  box-sizing: border-box;
  display: block;
