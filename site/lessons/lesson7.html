Lesson 5.1: CSS Layout Fundamentals

  • Normal document flow
  • Float and clear (legacy but important to understand)
  • Position property: static, relative, absolute, fixed

Example

.box {
  float: left;
  width: 100px;
  height: 100px;
  background: #1976d2;
  margin: 10px;
}
.clearfix {
  clear: both;
}
.relative {
  position: relative;
  left: 20px;
}
.absolute {
  position: absolute;
  top: 10px;
  left: 10px;
}
.fixed {
  position: fixed;
  bottom: 0;
  right: 0;
}

Lesson 5.2: Flexbox Layout

  • Flex containers and flex items
  • Main axis and cross axis
  • Flex properties: justify-content, align-items, flex-wrap
  • Common flexbox patterns

Example

.flex-container {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-wrap: wrap;
}
.flex-item {
  flex: 1 1 200px;
  margin: 10px;
  background: #90caf9;
}

Lesson 5.3: CSS Grid Basics

  • Grid containers and grid items
  • Grid template columns and rows
  • Grid areas and positioning
  • When to use Grid vs. Flexbox

Example

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-rows: 100px 100px;
  gap: 10px;
}
.grid-item {
  background: #e3f2fd;
  padding: 20px;
}
